# ml/ — эксперименты и обучение

Каталог для ML-составляющей SpectrumAI: датасеты, ноутбуки, скрипты разметки и
обучения. Не путать с `backend/app/ml/` — там продакшен-инференс.

## Структура

| Подкаталог | Назначение |
|---|---|
| `data/` | Сырые и обработанные датасеты (NIST/SDBS/Coblentz, синтетика). **В git не коммитится** (см. `.gitignore`). |
| `notebooks/` | Jupyter-ноутбуки: EDA, исследовательские эксперименты. |
| `scripts/` | CLI-скрипты: построение синтетического датасета, разметка через RDKit, обучение, экспорт весов. |
| `configs/` | YAML-конфиги гиперпараметров и пайплайнов. |
| `pipelines/` | Переиспользуемые модули: `Dataset`, augmentations, label utilities. |

## Двухфазный подход к данным (CLAUDE.md §11)

- **Фаза 1 (этапы 4–17):** работаем только на синтетическом датасете в
  `data/synthetic.parquet`. Метрики могут быть низкими — это нормально.
- **Фаза 2 (этапы 18+):** обучение на реальных данных, Kaggle T4/P100, целевые
  метрики из главы 6.4.4.

Не путать фазы. На фазе 1 не лезем в NIST.
