# Демо-спектры для предзащиты (v1.5-predefense)

Каталог для 5–10 JCAMP-DX-файлов, на которых обученная фазой 2 модель
показывает уверенные результаты. Спектры **отбираются вручную**
автором после прогона `02_predefense_train.ipynb`: загружаем тестовый
сабсет в систему через UI, оцениваем качество, выбираем удачные кейсы.

## Что положить

Файлы из `ml/data/raw/nist/<CAS>.jdx` (после `download_datasets.py`).
В каждом файле должно быть:
- валидный JCAMP-DX заголовок;
- покрытие 400–4000 см⁻¹ ≥ 80%;
- реальные интенсивности (не нули).

## Каталог требований к подбору

Цель — комиссия должна увидеть, что система работает. Поэтому:

1. **Чистые случаи (3–5 спектров):** известные соединения с одной-двумя
   ярко выраженными функциональными группами (этанол, ацетон, бензол,
   уксусная кислота, ацетонитрил). Модель должна на них уверенно
   выдавать top-1 = правильное соединение или совпадающие группы.
2. **Множественные группы (1–2 спектра):** соединения с 3–4 разными
   функциональными группами (например, метилбензоат — эфир +
   ароматическое кольцо). Демонстрирует, что классификатор не
   ограничен одной меткой.
3. **«Трудный» случай (1 спектр):** соединение, где top-1 не идеален,
   но top-5 содержит правильного кандидата. Это **полезно показать**
   комиссии — повод для конструктивной дискуссии о том, что ретривал
   страхует ошибки классификатора.

## Шаблон записи

Для каждого выбранного файла дополните таблицу ниже:

| Файл | Соединение | SMILES | Ожидаемые группы | Категория | Комментарий |
|------|------------|--------|------------------|-----------|-------------|
| `64-17-5.jdx` | ethanol | `CCO` | alcohol_OH, ch3_group, ch2_group | Чистый | Учебный кейс, top-1 точно правильный |
| `67-64-1.jdx` | acetone | `CC(=O)C` | ketone, ch3_group | Чистый | Сильный пик C=O 1715 см⁻¹ |
| `93-58-3.jdx` | methyl benzoate | `COC(=O)c1ccccc1` | ester, aromatic_ring, ch3_group | Множественные | Покажет multi-label силу |
| ...  | ...  | ...    | ...              | ...       | ...         |

> **После заполнения** обновите `docs/DEMO_SCRIPT.md` раздел «v1.5-predefense»,
> чтобы он ссылался на конкретные файлы из этой папки.

## Где взять CAS-номера

Скрипт `download_datasets.py --source nist_scrape --count 2000` уже
скачает их в `ml/data/raw/nist/`. Откройте `predefense_stats.json`,
выберите функциональные группы с высоким `positives_per_class`, найдите
соединения с этими группами через `Chem.MolFromSmiles` в Python.
