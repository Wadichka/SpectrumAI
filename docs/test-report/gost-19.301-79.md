## 4. Документы, на соответствие которым проводились испытания

## 5. Условия проведения испытаний

- Хост: Windows 11 Home 26200, x86_64, Docker Desktop 4.30+.
- Стек: `docker compose up -d` (PostgreSQL 16, Redis 7, FastAPI+gunicorn, nginx).
- Python 3.11.9; Node 20; Playwright Chromium 1.60.
- Сетевое окружение: localhost-only.

## 6. Перечень тестовых данных

- Синтетический JCAMP-DX `tests/integration/fixtures/sample.jdx` (этанол-подобный спектр);
- Каталог `backend/tests/integration/fixtures/` — типовые валидные спектры;
- Каталог `backend/tests/robustness/conftest.py` — программно сгенерированные битые спектры (NaN/Inf, искажения заголовка, кодировки);
- Локальная база `spectrumai-postgres` — пустая на момент испытаний (фаза 1).
