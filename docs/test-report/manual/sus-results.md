# Сводка результатов юзабилити-теста (SUS)

Шаблон для модератора. Заполняется **после** проведения сессий по
протоколу [`usability-protocol.md`](usability-protocol.md) с
3–5 респондентами целевой аудитории (студенты химических
специальностей).

Целевые пороги (NFR-07, глава 11 §11.6.6 пояснительной записки):
- **Приёмочный:** SUS ≥ 70 (медиана).
- **Целевой:** SUS ≥ 75 (медиана).
