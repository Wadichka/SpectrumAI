# SUS — System Usability Scale

Опросник для оценки удобства использования SpectrumAI согласно
главе 11 §11.6.6 пояснительной записки.

Шкала: 1 — категорически не согласен; 5 — полностью согласен.

Целевое значение SUS ≥ 70 (приёмочное), ≥ 75 (рекомендуемое).

## Расчёт балла

Для нечётных утверждений (1, 3, 5, 7, 9): балл = (ответ − 1).
Для чётных утверждений (2, 4, 6, 8, 10): балл = (5 − ответ).
Сумма баллов умножается на 2,5 — итог в диапазоне 0–100.
